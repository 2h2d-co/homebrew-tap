# 2h2d-co Homebrew Tap

Homebrew tap for [2h2d.co](https://www.2h2d.co) tools.

## Install

Install formulas directly from the tap:

```sh
brew install 2h2d-co/tap/cage
brew install 2h2d-co/tap/dots
brew install 2h2d-co/tap/brew-safe
```

## Formulas

- `cage` — minimal macOS CLI for loading 1Password Environments with age-protected 1Password service account tokens.
- `dots` — minimal, copy-based dotfiles manager.
- `brew-safe` — Homebrew commands for release-age-gated upgrades.
